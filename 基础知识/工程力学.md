# 📚 研究生补课笔记：工程静力学 (Statics)

**适用对象：** 地质灾害方向研究生（材料学背景）

**核心目标：** 掌握地质体（岩土）的受力平衡与失稳判据

---

## 1. 力的基本运算：地质体的受力拆解

在地质灾害中，力通常不是垂直或水平的，必须熟练掌握**矢量分解**。

### 1.1 力的正交分解 (Resolution of Forces)

- **场景：** 分析斜坡上的土体。
    
- **模型：** 重力 $W$，坡角 $\alpha$。
    
- **核心分解：**
    
    - **下滑力 (Driving Force):**
        
        $$F_x = W \sin \alpha$$
        
        - _性质：_ “破坏力”，试图让山体滑下去。
            
    - **压坡力 (Normal Force):**
        
        $$F_y = W \cos \alpha$$
        
        - _性质：_ “稳定力”，通过产生摩擦来按住山体。
            

### 1.2 力矩 (Moment)

- **定义：** 力使物体绕点转动的趋势。公式：$M = F \times d$。
    
- **地质应用：危岩体倾倒 (Toppling)**
    
    - 当重心产生的翻转力矩 > 抗倾覆力矩时，灾害发生。
        
    - _注意：_ 许多崩塌不是“滑”下来的，是“翻”下来的。
        

---

## 2. 受力分析：滑坡计算的核心 (FBD)

**受力分析图 (Free Body Diagram)** 是计算安全系数的绝对前提。

### 2.1 隔离体法 (Method of Isolation)

- **定义：** 将研究对象（滑块/危岩）从周围环境中取出，单独画出所有受力。
    
- **地质特有的力（材料学较少涉及）：**
    
    - **孔隙水压力 (Pore Water Pressure):** 水对岩石内部孔隙的压力，表现为浮力或渗透力。
        
    - **地震力 (Seismic Force):** 通常简化为水平方向的惯性力 ($F = ma$)。
        

### 2.2 约束与反力 (Constraints)

|**约束类型**|**对应地质结构**|**反力特征**|
|---|---|---|
|**柔性约束**|锚索、锚杆|只有拉力 (Tension)|
|**光滑接触**|泥化夹层、断层泥|只有法向支持力 ($N$)，摩擦力 $\approx 0$|
|**粗糙接触**|正常潜在滑动面|法向支持力 ($N$) + 摩擦力 ($f$)|

---

## 3. 平衡方程：上帝的法则

对于二维平面问题（如边坡剖面），物体静止必须同时满足三个方程：

$$\begin{cases} \sum F_x = 0 & \text{(水平方向合力为零)} \\ \sum F_y = 0 & \text{(垂直方向合力为零)} \\ \sum M_O = 0 & \text{(力矩代数和为零)} \end{cases}$$

> **导师批注：** 滑坡稳定性分析（如毕肖普法 Bishop's Method）的本质，就是建立极限平衡状态下的方程组，求解安全系数 $F_s$。

---

## 4. 摩擦定律：滑坡的刹车片

这是连接“静力学”与“岩石材料性质”的桥梁。

### 4.1 库仑摩擦定律 (Coulomb Friction)

- **物理公式：**
    
    $$F_{max} = \mu N$$
    
- 地质公式 (莫尔-库仑准则)：
    
    $$\tau = c + \sigma \tan \phi$$
    
    - $\tau$: 抗剪强度
        
    - $c$: 粘聚力 (Cohesion) - 岩石内部的胶结力
        
    - $\phi$: 内摩擦角 (Friction Angle) - 颗粒间的咬合力
        

### 4.2 自锁现象 (Self-locking)

- 若斜坡角度 $\alpha < \phi$（内摩擦角），无论滑块多重，仅靠重力**永远不会**滑动。
    
- _启示：_ 测准 $\phi$ 值是评价边坡是否自然稳定的关键。

## 5. 分布力的简化：水压力

土体和水压力不是作用在一个点上，而是分布在面上。

### 5.1 静水压力三角形

- **分布规律：** 压强随深度线性增加，$P = \rho g h$。
    
- **合力作用点：**
    
    - 对于三角形分布荷载，合力作用点位于底部往上 **1/3 高度处**。
        
    - _实战：_ 计算泥石流冲击拦挡坝时，必须知道合力作用点才能算出力矩，防止坝体被推倒。
    - （1）合力形状
```
        水面 (Water Level)  h = 0, P = 0
    ↓
    +------------------------- 墙顶
    |
    | .                        (压力极小)
    | ->
    |
    | .
    | --->                     (压力中等)
    |
    | .
    | ----->
    |
    | .
    | ------->                 (压力较大)
    |
    | .
    | --------->               (底部压力最大)
    +------------------------- 墙底 (h = H)
      ^
      |
    最大压强 P_max = ρgh
```

- （2）合力大小
- **三角形面积公式：** $\text{Area} = \frac{1}{2} \times \text{底} \times \text{高}$
    
- 代入物理量：
    
    $$F = \frac{1}{2} \times P_{max} \times H$$
    
    $$F = \frac{1}{2} \times (\rho g H) \times H = \frac{1}{2} \rho g H^2$$
    
- **重点：** 水推力与水深的**平方**成正比！
    
    - 水深变成 2 倍，推力变成 **4 倍**。这就是为什么冰湖水位稍微涨一点点，大坝就可能瞬间崩溃。
- 
    - （3）合力作用点
```|
          |
          |      /|
          |    /  |
          |  /    |
 水深 H   |/      |
          |       |
          |   F   | <-- 合力作用点 (重心)
          |<------|
          |       | 
          |_______|
          
             ↑
        距离底部 H/3
```

## 💡 导师的实战思考题：为什么下雨会导致滑坡？

结合静力学知识，我们可以给出一个完美的物理学解释。

1. **模型：** 斜坡上的滑块。
    
2. **未下雨时：** 支持力 $N = W \cos \alpha$。最大摩擦力 $f = \mu W \cos \alpha$。
    
3. **下雨后（地下水位上升）：**
    
    - 水对滑块底部产生向上的浮力（孔隙水压力）$U$。
        
    - 垂直方向平衡方程变为：
        
        $$N + U = W \cos \alpha \quad \Rightarrow \quad N = W \cos \alpha - U$$
        
4. **结论：**
    
    - 有效支持力 $N$ **减小**。
        
    - 最大抗滑力（摩擦力） $f_{max} = \mu N$ 随之**剧烈减小**。
        
    - 当 $f_{max}$ 小于下滑力 $W \sin \alpha$ 时，滑坡发生。
        

> **一句话总结：** 水并没有“润滑”岩石，而是通过浮力“托”起了岩石，卸掉了它的“刹车”能力（有效正应力）。

# 📚 核心重筑：材料力学 (Mechanics of Materials) 详解版

核心逻辑：

材料力学研究三个基本问题：

1. **强度 (Strength)：**会不会断？（比如：滑坡会不会启动？）
    
2. **刚度 (Stiffness)：**会不会变形太大？（比如：地面沉降会不会把房子拉裂？）
    
3. **稳定性 (Stability)：**会不会失稳压溃？（比如：细长的抗滑桩会不会被压弯？）
    

---

## 1. 基本概念：内力的“微观”表达

外力（重力、地震力）作用在物体上，物体内部会产生对抗的力，这就是**内力**。为了去除物体尺寸的影响，我们用**应力**来描述。

### 1.1 正应力 (Normal Stress, $\sigma$)

- **定义：** 垂直于截面的力。
    
- **形象理解：**
    
    - **压应力 (Compression)：** 像把弹簧压扁。**在地质学中，规定受压为正 (+)**（因为地球深处全是高压）。
        
    - **拉应力 (Tension)：** 像把拉面拉长。规定受拉为负 (-)。
        
- **地质痛点：** 岩石和土体**极度怕拉**。岩石的抗压强度通常是抗拉强度的 10 倍以上。一旦出现拉应力（比如在坡顶），就会出现张拉裂缝，雨水灌入，灾害开始。
    

### 1.2 剪应力 (Shear Stress, $\tau$)

- **定义：** 平行于截面的力。
    
- **形象理解：** 像用剪刀剪纸，或者扑克牌被搓开。
    
- **地质痛点：** **这是万恶之源。** 99% 的滑坡、泥石流、断层错动，本质上都是因为**剪应力 $\tau$ 超过了材料的抗剪能力**。
    

### 1.3 应变 (Strain, $\varepsilon$)

- **定义：** 描述变形的程度（单位长度的变形量）。
    
- 胡克定律 (Hooke's Law)：
    
    在弹性范围内，应力与应变成正比：
    
    $$\sigma = E \cdot \varepsilon$$
    
    - **$E$ (杨氏模量/弹性模量)：** 代表材料的**刚度**（硬度）。$E$ 越大，材料越不容易变形（如花岗岩）；$E$ 越小，材料越软（如粘土）。

## 2. 复杂受力状态：一点处的应力分析

以前你做实验可能只是拉一根钢筋（单向受力），但山体里的岩石是受到四面八方挤压的（三向受力）。

### 2.1 主平面与主应力

想象在岩石内部切无数刀，总能找到三个相互垂直的面，这些面上**只有正应力，没有剪应力**。

- **$\sigma_1$ (大主应力)：** 最大的那个压力。
    
- **$\sigma_3$ (小主应力)：** 最小的那个压力（通常是围压）。
    
- **$\sigma_2$ (中主应力)：** 中间的（平面问题中常忽略）。
    

### 2.2 莫尔圆 (Mohr's Circle) —— 图解计算器

如果你忘了公式，必须记住这个图。这是一个能求出**任意角度截面上**应力的几何工具。

- **怎么画：**
    
    1. 在横轴（$\sigma$轴）上标出 $\sigma_3$ 和 $\sigma_1$。
        
    2. 以此为直径画一个圆。
        
- **怎么看：**
    
    - 圆周上的每一个点，都代表岩石内部某一个特定角度切面上的应力状态 ($\sigma, \tau$)。
        
    - **圆顶最高点：** 代表**最大剪应力**，数值为 $\tau_{max} = \frac{\sigma_1 - \sigma_3}{2}$。
        
- **地质意义：** 如果这个圆大到碰到了“破坏包络线”，岩石就碎了。
    

---

## 3. 强度理论：岩石是怎么“死”的？

这是你需要“死记硬背”并深刻理解的部分。

### 3.1 为什么不用第四强度理论（Von Mises）？

材料系学金属常用 Von Mises（形状改变比能理论），认为材料屈服是因为变形能太高。但**岩土体是颗粒材料**，它的破坏主要靠摩擦失效。

### 3.2 摩尔-库仑理论 (Mohr-Coulomb) —— 地质界的“牛顿定律”

岩土体的破坏主要取决于**剪应力**，但这个能承受的最大剪应力不是常数，它会随着压力的增加而变大。

公式详解：

$$\tau_f = c + \sigma \tan \phi$$

1. **$\tau_f$ (抗剪强度)：** 破坏瞬间的剪应力。
    
2. **$\sigma$ (正应力)：** 面上受到的压紧力。压得越紧，越难剪断（摩擦力越大）。
    
3. **$c$ (粘聚力 Cohesion)：**
    
    - **解释：** 材料原本的“粘性”。
        
    - **例子：** 湿润的粘土捏成团，$c$ 值高；干沙子散开，$c \approx 0$。
        
4. **$\phi$ (内摩擦角 Friction Angle)：**
    
    - **解释：** 材料颗粒粗糙度和咬合度。
        
    - **例子：** 碎石堆的坡度通常就是它的内摩擦角。

## 4. 弯曲与扭转：抗滑桩的受力

在治理滑坡时，我们会打入钢筋混凝土桩（抗滑桩）。这时候，你需要把桩看作一根“梁”。

### 4.1 弯矩 (Bending Moment, $M$)

- **场景：** 滑坡推力推在桩背上，试图把桩折断。
    
- **中性轴 (Neutral Axis)：** 梁弯曲时，中间有一层既不拉也不压。
    
- **受力分布：**
    
    - 受拉区（通常是桩的背面）：混凝土怕拉，所以这里必须放大量**钢筋**。
        
    - 受压区（通常是桩的正面）：混凝土不怕压。
        

### 4.2 剪力 (Shear Force, $V$)

- **场景：** 滑坡推力试图把桩像切黄瓜一样直接剪断。
    
- **危险截面：** 通常在**滑动面**的位置，剪力最大。

## 5. 材料学背景的“转化指南”

为了帮你快速唤醒记忆，我做了一个**概念映射表**：

|**材料学概念 (你的旧知识)**|**地质力学/岩土力学 (你的新战场)**|**核心区别**|
|---|---|---|
|**拉伸试验** (Tensile Test)|**三轴压缩试验** (Triaxial Test)|岩石在地下是被高压围住的，必须加围压 ($\sigma_3$) 做实验。|
|**屈服点** (Yield Point)|**峰值强度** (Peak Strength)|岩石破坏往往是脆性的，过了峰值瞬间崩塌。|
|**晶界/位错** (Defects)|**节理/断层** (Joints/Faults)|尺度不同。地质灾害中，弱面（裂缝）决定整体强度，而不是岩石本身。|
|**应力集中** (Stress Conc.)|**应力集中**|原理一样。断层尖端、隧道周边容易应力集中导致岩爆。|
|**各向同性** (Isotropic)|**各向异性** (Anisotropic)|页岩、片麻岩有层理，横着压和竖着压强度完全不同。|

## 💡 老师给你的复习建议

你不需要去重新推导公式。你需要做的是建立**物理直觉**。

**请思考以下场景（用刚学的知识）：**

> 为什么在山脚开挖（修路、盖房）容易引发山顶滑坡？

**答案解析：**

1. 开挖切除了坡脚，相当于移除了部分**抗滑力**。
    
2. 坡体内部应力重分布，**剪应力 $\tau$** 增大。
    
3. 同时，由于临空面出现，侧向约束 $\sigma_3$ 减小（甚至变成 0）。
    
4. 根据莫尔圆，$\sigma_3$ 变小，莫尔圆半径 $(\sigma_1 - \sigma_3)/2$ 变大（最大剪应力变大）。
    
5. 圆碰到了莫尔-库仑包络线 $\rightarrow$ **破坏发生**。
    

这就是你学习材料力学的最终目的：**看穿山体破坏背后的力学机制。**
